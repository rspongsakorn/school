B — Full Sidebar (เหมือน Desktop)

แสดง sidebar เต็มรูปแบบ 260px เหมือนหน้าจอ desktop เนื้อหาจะแคบลงแต่เห็น navigation ครบ

C — เหมือน Mobile (Drawer)

Tablet ใช้ pattern เดียวกับ mobile ทั้งหมด — hamburger + drawer เนื้อหาเต็มจอ layout จะเรียบง่าย

คลิกเลือกใน browser หรือบอกใน terminal ได้เลยครับ (A = mini icon sidebar, B = full sidebar, C = เหมือน mobile)
